如何使用 MacPorts 安装 ntfs-3g 并挂载 NTFS 外接硬盘

安装 ntfs-3g：

  1. 安装 Xcode：从 Mac App Store 下载并安装 Xcode（如果尚未安装）。Xcode 是 Apple 提供的一套开发工具，包括编译器和开发实用程序。
  2. 安装 Xcode 命令行工具：在终端中输入以下命令以安装 Xcode 命令行工具： 
    xcode-select --install
  3. 安装 MacPorts：访问 MacPorts 官方网站（https://www.macports.org/install.php），根据您的 macOS 版本下载并安装对应的 MacPorts 安装包。
  4. 更新 MacPorts：在终端中输入以下命令以更新 MacPorts 及其可用软件列表： 
    sudo port selfupdate
  5. 安装 ntfs-3g：在终端中输入以下命令，使用 MacPorts 安装 ntfs-3g： 
    sudo port install ntfs-3g

挂载 NTFS 外接硬盘：

  1. 将 NTFS 格式的外接硬盘连接到您的 Mac 电脑上。
  2. 打开终端，输入以下命令以查找已连接的 NTFS 卷的设备名称和挂载点： 
    diskutil list
    在输出中找到您的 NTFS 卷所在的设备（例如，/dev/disk2s1）和挂载点（例如，/Volumes/MyNTFSVolume）。
  3. 卸载 NTFS 卷。用实际的设备名称替换 <YOUR_NTFS_DEVICE>，然后在终端中输入以下命令： 
    sudo diskutil unmount /dev/<YOUR_NTFS_DEVICE>
    例如，如果您的 NTFS 设备名称为 /dev/disk2s1，请输入 sudo diskutil unmount /dev/disk2s1。
  4. 使用 ntfs-3g 重新挂载 NTFS 卷。将 <YOUR_NTFS_DEVICE> 替换为实际的设备名称，将 <YOUR_NTFS_MOUNT_POINT> 替换为实际的挂载点，然后在终端中输入以下命令： 
    sudo ntfs-3g /dev/<YOUR_NTFS_DEVICE> <YOUR_NTFS_MOUNT_POINT>
    例如，如果您的 NTFS 设备名称为 /dev/disk2s1，挂载点为 /Volumes/MyNTFSVolume，请输入 sudo ntfs-3g /dev/disk2s1 /Volumes/MyNTFSVolume。
